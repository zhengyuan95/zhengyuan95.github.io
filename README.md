# zhengyuan95.github.io

Personal academic homepage for Zhengyuan Zhang, Postdoctoral Research Fellow at Nanyang Technological University.

Live at: https://zhengyuan95.github.io

## Structure

- `index.html` — single-page site (about, education, experience, research, publications, awards, teaching, skills)
- `style.css` — styling (light/dark mode aware)

## Updating

Edit `index.html` / `style.css` directly and push to `main`; GitHub Pages redeploys automatically within a minute or two.
